Features
Built for precision
Displacement Textures
Apply built-in or custom displacement maps to any STL mesh. Amplitude set in millimetres — no unitless coefficients.
Adaptive Subdivision
Control mesh density by max edge length (mm). Fine enough for high-frequency textures, coarse enough for your slicer.
Real-Time Preview
3D viewport powered by Three.js. See exactly how your mesh will look before you export.
Symmetric Mode
Mirror displacement for perfectly symmetric textures. Essential for organic forms and repeating patterns.
Clean STL Export
Exports watertight meshes with no holes, no T-junctions. Ready to open in OrcaSlicer or any other tool.
Native Desktop App
Built with Tauri — tiny binary, native performance, no Electron overhead. Works fully offline.
Workflow
Three steps
01
Load your mesh
Drag and drop any STL file. SurfEx parses it instantly.
02
Configure texture
Pick a displacement map, set amplitude in mm and edge length. Preview updates live.
03
Export & print
Export a clean, watertight STL. Open in your slicer and print.
